# bysj

#### 介绍
老陆的毕设

#### 软件架构
软件架构说明
前端使用 vue + element-ui
后端使用 springboot + spring security + reids

#### 安装教程

1.  先安装redis
2.  运行数据库脚本（数据库名bysj)
2.  运行后端代码（bysj)
3.  运行前端代码

#### 使用说明

1.  分布式系统，前端请在三台服务器上部署，不然可能会出现问题
